## 测试步骤

### 测试输出变量显示：
1. 打开浏览器控制台（F12）
2. 查看节点1，应该显示：
   - `@output sum: 80`（50+30）
   - `@output product: 1500`（50*30）
3. 调整x滑动条，观察输出值变化

### 测试中键连接：
1. 按住鼠标中键点击节点1
2. 控制台应显示：`中键点击，当前连接状态: null` 和 `设置连接起始节点: 1`
3. 再按住中键点击节点3
4. 控制台应显示连接创建信息
5. 应该看到从节点1到节点3的连接线

### 测试日志显示：
1. 节点1应显示：
   ```
   log---------------
   计算结果: sum=80
   product=1500
   ```
